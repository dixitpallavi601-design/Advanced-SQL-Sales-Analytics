# Advanced SQL Sales Analytics

## 📌 Project Overview
This project analyzes sales data using SQL Server to uncover business insights and trends.

## 🎯 Objectives
- Perform Exploratory Data Analysis (EDA)
- Analyze monthly and yearly sales trends
- Calculate cumulative sales
- Compare product performance year over year
- Segment customers based on spending behavior
- Analyze category contribution to overall sales

## 🛠️ SQL Concepts Used
- Joins
- Common Table Expressions (CTEs)
- Window Functions
- Aggregate Functions
- CASE Statements
- GROUP BY
- HAVING
- Date Functions
- DATEDIFF
- DATETRUNC

## 📊 Analysis Performed
- Exploratory Data Analysis
- Monthly Sales Trends
- Yearly Sales Trends
- Running Total Analysis
- Moving Average
- Product Performance Analysis
- Part-to-Whole Analysis
- Customer Segmentation
